BRAND / LOGO FILES
==================
  logo.png        header logo (colour, transparent background)
  logo-white.png  footer logo (white version, for the dark footer)
  emblem.png      browser-tab icon (favicon)

TO CHANGE THE LOGO
------------------
Replace these files, keeping the EXACT same names.
- Use PNG files with a transparent background.
- logo-white.png must be an all-white version so it shows on the dark footer.
If the shapes differ a lot from the current ones, the header spacing may need
a tweak - ask whoever set this up if it looks off.
