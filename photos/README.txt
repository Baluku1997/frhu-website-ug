HERO PHOTO + SLIDER PHOTOS
==========================

THE HERO PHOTO (top-right card on the landing page)
---------------------------------------------------
  hero.jpg   Drop a strong LANDSCAPE photo here (about 1200px wide or more).
             It appears behind the navy "Our approach" card.
             If there is no hero.jpg, the card just stays plain navy - fine.

THE SLIDING BANNER (very top of the page)
-----------------------------------------
  activity-1.jpg
  activity-2.jpg
  activity-3.jpg
  activity-4.jpg
These four slide across automatically. Use wide landscape photos, 1600px+.

TO CHANGE ANY OF THESE
----------------------
Drop in a new file with the SAME name (lowercase, .jpg).
Want different captions on the banner, or more than 4 slides? That's a small
edit inside index.html (search for: const SLIDES) - or ask for help.
