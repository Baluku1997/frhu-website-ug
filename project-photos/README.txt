PROJECT PHOTOS
==============

  floodadapt-cover.jpg    The round photo on the project card in the carousel.
                          Use a SQUARE, subject-centred photo (600x600 or more).

  floodadapt-1.jpg ... floodadapt-14.jpg
                          The photos inside the FloodAdapt gallery, in order.

TO CHANGE A PHOTO
-----------------
Drop in a new file with the SAME name (e.g. replace floodadapt-3.jpg).

TO ADD OR REMOVE GALLERY PHOTOS
-------------------------------
Add e.g. floodadapt-15.jpg here, then add one line in index.html
(search for: project-gallery) copying an existing image line. Or ask for help.

TO ADD A WHOLE NEW PROJECT
--------------------------
Each project is a card + a hidden detail block in index.html. This is a bigger
edit - the easiest path is to send the new project's photo, title, short blurb,
stats and gallery photos to whoever set this up, and it slots in next to
FloodAdapt.
