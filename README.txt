FRONT RISE HEALTH UGANDA - WEBSITE
==================================

HOW TO VIEW THE SITE
--------------------
Keep this whole folder together. Double-click "index.html" to open the site
in your web browser. The index.html file and these folders must always stay
side by side - if you move index.html on its own, the images stop showing.

WHAT'S IN HERE
--------------
  index.html      the website itself
  brand/          the logo files
  photos/         the hero photo + the sliding banner photos
  theme-photos/   the 6 "What we do" theme photos
  team-photos/    the 8 team headshots
  project-photos/ the FloodAdapt project cover + gallery photos
  partners/       the funder / partner logos

Each folder has its own README.txt telling you exactly what to put there
and how to change it.

THE GOLDEN RULE FOR PHOTOS
--------------------------
To change any photo, drop in a new file with the EXACT same name as the old
one (same spelling, same .jpg). Names are case-sensitive: hero.jpg works,
Hero.jpg or hero.JPG do NOT. Some phones save a capital .JPG - rename it to
lowercase .jpg.

NEED A BIGGER CHANGE?
---------------------
Adding a whole new project, changing wording, colours, or layout means
editing index.html. If you're not comfortable doing that, send the details
(and any photos) to whoever set this up and it can be done for you quickly.
